## 🔧 配置修改

### 查看配置
```bash
adb shell su -c "cat /data/adb/modules/AppOpt_8Gen2_Mi13/applist.conf | head -20"
```

### 修改配置
```bash
# 1. 拉取配置文件
adb pull /data/adb/modules/AppOpt_8Gen2_Mi13/applist.conf

# 2. 编辑配置 (使用文本编辑器)
# 格式: 包名=CPU核心范围
# 例如: com.tencent.mm=0-2

# 3. 推送回设备
adb push applist.conf /data/adb/modules/AppOpt_8Gen2_Mi13/

# 4. 重启AppOpt (配置即时生效)
adb shell su -c "killall AppOpt"
adb shell su -c "/data/adb/modules/AppOpt_8Gen2_Mi13/AppOpt &"
```

### 配置示例
```conf
# 游戏优化 - 主进程小核，渲染大核
com.tencent.tmgp.sgame=0-2
com.tencent.tmgp.sgame{UnityMain*}=6-7

# 日常应用 - 主进程小核，渲染中核
com.tencent.mm=0-2
com.tencent.mm{RenderThread}=3-4

# 省电模式 - 全部小核
com.某应用=0-2
```
