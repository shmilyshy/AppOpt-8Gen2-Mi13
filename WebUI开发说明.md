# AppOpt WebUI 开发说明

## 概述
已完成AppOpt WebUI的完整代码框架，这是一个基于C语言的轻量级Web管理界面，用于可视化管理骁龙8Gen2小米13的线程优化配置。

**作者**: 刘十三  
**开发语言**: C语言  
**目标平台**: Android ARM64  
**预期大小**: ~500KB单一二进制

## 已完成内容

### ✅ 代码实现 (100%)
1. **HTTP服务器** (src/http_server.c)
   - 基于libmicrohttpd
   - 支持GET/POST请求
   - 静态文件服务
   - JSON响应处理

2. **API接口** (src/api.c)
   - `/api/status` - 系统状态
   - `/api/apps` - 应用列表
   - `/api/config` - 配置管理
   - `/api/save` - 保存配置
   - `/api/presets` - 预设方案

3. **配置管理** (src/config.c)
   - 读取applist.conf
   - 解析线程绑定规则
   - 写入配置文件

4. **应用管理** (src/app_manager.c)
   - 获取应用列表
   - 查询运行状态
   - 读取CPU亲和性
   - Shell命令执行

5. **前端界面** (web/)
   - 响应式设计
   - 实时状态显示
   - 应用列表展示
   - 自动刷新

## 技术架构

### 后端 (C语言)
```
libmicrohttpd (HTTP服务器)
    ↓
http_server.c (请求路由)
    ↓
api.c (业务逻辑)
    ↓
config.c + app_manager.c (数据处理)
    ↓
applist.conf (配置文件)
```

### 前端 (嵌入式)
```
index.html (页面结构)
    ↓
style.css (样式设计)
    ↓
app.js (交互逻辑)
    ↓
API接口 (数据交互)
```

## 使用方法

### 编译
```bash
cd webui

# 设置NDK路径
export NDK_PATH=/path/to/android-ndk

# 编译
./build.sh
# 或
make ARCH=arm64
```

### 部署
```bash
# 推送到设备
adb push bin/AppOptWebUI /data/adb/modules/AppOpt_8Gen2_Mi13/
adb shell su -c 'chmod 755 /data/adb/modules/AppOpt_8Gen2_Mi13/AppOptWebUI'

# 启动服务
adb shell su -c '/data/adb/modules/AppOpt_8Gen2_Mi13/AppOptWebUI -d'
```

### 访问
```
浏览器打开: http://手机IP:8080
```
